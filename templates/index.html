Water Wheels

An inventory management tracking system that focuses on productivity, transparency, and security.


Productivity

The product systems allows the user to create, delete, and update items in the database. The list of items are also displayed within the same page. Filling out the forms will generate the product.

Along with the database, there is a graph that displays which products are in stock in ascending order of what is in stock. This allows the user to visualize the data to quickly find which products are low in stock.

A page that displays a calendar of events. The calendar is able to show different events across multiple years while having the capability of adding new ones.

Transparency

Actions a user takes within the database and admin features will be recorded for reference.

With a registered employee ID, a user can input their ID to clock-in and clock-out. There is a table that displays when the user clocked-in and clocked-out along with their name and employee ID.

Security

A page that requires admin access to see the user database. A user can register for admin access by using an admin code. The admin feature restricts users from deleting users and other important data.
